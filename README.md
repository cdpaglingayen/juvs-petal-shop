# JUVS PETAL SHOP

## Instructions

### File Structure

1. Remove `.example` in `.env.example` or create your own `.env` file.

### Terminal

2. Run the following commands:
   - `composer i`
   - `npm i`
   - `php artisan key:generate`
   - `npm run dev`
